OVERVIEW

Swoop is a full stack web application that allows teams to assign tasks and write informative articles about the project at hand. Organizations often use custom built project management systems and content management systems but, it is rare to find both of these systems combined into one. Which is why I thought it would be useful to build this tool for my web application engineering final project.


FEATURES

Content Management Functionalities

This application's content management features leverage markdown technology to produce easy to write articles about any sort of project. Users can embed images, videos and more to produce useful "how to" styled articles.


Project Management System

Swoop takes a simplistic approach to task management by allowing the user to set reachable deadlines for themselves and their team members.


Cloud Storage Capabilities

Users are able to upload images and documents to SCTMS. Users can associate their files to a specific article that was created using our CMS or simply associate files to a particular project.
